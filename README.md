# Staked Aura

A DeFi staking platform built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Build

```sh
# Build for production
npm run build
```
